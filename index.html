// MECHANISM · 02 — // HOW IT WORKS
02

Scan. Score.
Signal.

SPECTER runs continuously on Aeon's GitHub Actions infrastructure. No servers. No exposed keys. Hermes memory layer accumulates behavioral data across every session — building a cryptographic identity that compounds over time.

Aeon · Scheduled
Continuous On-Chain Audit

Every 15 minutes, Aeon's heartbeat scans the agent's full transaction history — counterparty diversity, contract interactions, debt repayment patterns, and flagged protocol exposure. Zero infrastructure required.

Hermes · Memory
7-Dimensional Behavioral Vector

Hermes accumulates cross-session behavioral data into a persistent 7-dimensional vector. Each dimension self-improves — one agent's lesson becomes every agent's upgrade.

ERC-8004 · On-Chain
Cryptographic Trust Proof

The score is written on-chain via ERC-8004 — a cryptographically signed identity passport readable by any protocol. DeFi pools, commerce brokers, and credit systems query SPECTER to gate access and price risk.
