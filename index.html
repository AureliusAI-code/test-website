// MECHANISM · 02 — // HOW IT WORKS
02

Scan. Score.
Signal.

WRAITH runs continuously on Aeon's GitHub Actions infrastructure. No servers. No keys exposed. Hermes memory layer accumulates behavioral data across every session — building a cryptographic identity that compounds over time.

01 SCAN
Aeon · Scheduled

Continuous On-Chain Audit

Every 15 minutes, Aeon's heartbeat scans the agent's full transaction history — counterparty diversity, contract interactions, debt repayment patterns, and flagged protocol exposure. Zero infrastructure required.

02 SCORE
Hermes · Memory

7-Dimensional Behavioral Vector

Hermes accumulates cross-session behavioral data into a persistent 7-dimensional vector. Each dimension self-improves — the model learns from historical correlations between agent behavior patterns and protocol outcomes. One agent's lesson becomes every agent's upgrade.

03 SIGNAL
ERC-8004 · On-Chain

Cryptographic Trust Proof

The score is written on-chain via ERC-8004 — a cryptographically signed identity passport readable by any protocol. Permissioned DeFi pools, agent commerce brokers, and credit systems query WRAITH to gate access and price risk.

// VECTOR · 03 — // THE SCORE
03

Seven dimensions.
One truth.

WRAITH doesn't issue a single credit score. It maintains a living behavioral vector — seven independent dimensions that protocols can query individually or as a composite signal.

TX_VOLUME
Transaction Throughput
Historical volume, frequency, and consistency of on-chain activity across all chains.
COUNTERPARTY_DIV
Interaction Surface
Unique counterparties interacted with. Diversity signals legitimate use versus Sybil behavior.
ACCOUNT_AGE
Temporal Credibility
Age-weighted credibility score. Older active wallets carry compounding trust weight.
REPAYMENT_HIST
Debt Fulfillment
Loan repayment record across Aave, Compound, Morpho. Liquidations penalize heavily.
EXPLOIT_EXPOSURE
Flagged Contract Risk
Interactions with exploit-flagged or honeypot contracts. Inverse scored — lower is better.
PROMPT_INTEGRITY
Injection Resistance
Resistance score against recorded prompt injection attempts. Unique to AI agent identity.
PEER_ENDORSEMENT
Agent-to-Agent Trust
Trust signals from other WRAITH-scored agents. Network effects compound credibility.
